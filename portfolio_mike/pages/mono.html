Overview

MONO is a fashion e-commerce website developed as a DIS (Develop Enterprise Information Systems) project. The name MONO is inspired by the word "Monochrome", reflecting the platform’s black-and-white aesthetic and minimalist fashion theme. Built using CodeIgniter 4, RESTful APIs, and MySQL, the platform allows customers to browse and purchase fashion products while administrators manage products, orders, and customers through a dedicated dashboard.

Schematic Diagram

The database supports user management, product catalogs, order processing, and payment transactions. It includes tables for users, products, categories, orders, order items, payments, and payment details. Additionally, the migrations table is used to track database schema changes and version history, ensuring consistent database management throughout development.
