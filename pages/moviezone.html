Overview

MoviZone is a movie discovery application inspired by the words "Movie" and "Zone", representing a place where users can explore a wide variety of films. Built using Flutter, PHP, and MySQL, the application integrates the TMDB API to provide up-to-date movie information, including ratings, descriptions, posters, and other movie details.

Features

lock

Authentication

Register a new account, log in, and log out securely with session-based authentication.

filter_list

Filter by Category

Browse movies filtered by genre and category for a more focused discovery experience.

search

Search Movies

Search for any movie by title using the TMDB API with real-time results and recent search history.

favorite

Favorite Movies

Add movies to your personal favorites list and remove them whenever you like.

manage_accounts

Edit Profile

Update your username and manage your profile photo add or delete it as needed.

movie

Movie Detail

View full movie details including trailer, overview, cast & crew, pulled live from TMDB API.

Information Architecture

The app's navigation is structured around a bottom navigation bar with four main tabs: Home, Explore, Favorite, and Profile. All flows originate from the authentication screens, and any movie card across all tabs navigates to the shared Detail Movie screen.
