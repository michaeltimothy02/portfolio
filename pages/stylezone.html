Overview

StyleZone is a fashion e-commerce website developed as a WAP (Web Application Programming) project. The name combines "Style", representing fashion and self-expression, and "Zone", representing a place where various fashion styles come together. Built with PHP and MySQL, the platform allows customers to browse and purchase products while enabling administrators to manage the store through a dedicated dashboard.

Database Design

The database is structured to support the full e-commerce workflow from user authentication and product catalog management to order processing and admin controls. Key tables include users, products, categories, orders, order_items, payments, and payment_details, all connected through relational foreign keys to maintain data integrity.
